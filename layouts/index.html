The Lifetime Financial Obligation (LFO) Calculator was built to help Washington State judges and justices adjudicate cases involving LFOs. The Calculator allows users to add all the charges as well as financial and legal conditions of the defendant to understand the impact of different sentencing options, including fines, community-service hours, probation, and more.

Try the LFO Calculator

Mobile-first Design

The LFO Calculator uses a modern web design framework that allows it work in all kinds of devices.

Cross-browser

The LFO Calculator works in all modern browsers, both desktop and mobile, including the latest Chrome, Firefox, Safari, Opera, IE (> 11) and Edge versions.

Open Source

The LFO Calculator is released under the MIT Open Source License and it free to use for personal and commercial purposes.
