The Legal Financial Obligation (LFO) Calculator was built to help Washington State judges adjudicate cases involving LFOs. The calculator provides users the only centralized resource combing laws specific to the charged crimes, a personâ€™s ability to pay, and tools to determine reasonable payment options. The Washington State Supreme Court's Minority and Justice Commission created the LFO calculator as part of a Department of Justice grant under a program called "Price of Justice: Rethinking the Consequences of Justice Fines and Fees." Microsoft joined the effort as part of their Corporate Responsibility mission, bringing technical resources and funding to the table in order to build and deliver this as a publicly available resource and bring awareness to LFOs.

Mobile-first Design

The LFO Calculator uses a modern web design framework that allows it work in all kinds of devices.

Cross-browser

The LFO Calculator works in all modern browsers, both desktop and mobile, including the latest Chrome, Firefox, Safari, Opera, IE (> 11) and Edge versions.

Open Source

The LFO Calculator is released under the MIT Open Source License and it free to use for personal and commercial purposes.
